# A Cronjob Monitor
The monitor sends a heartbeat to Cronitor

## Installation
```
composer require lotsofpixels/module-cronitor-magento2
```
## Version
Release 1.0.0

## Configuration
Stores -> Configuration -> Lotsofpixels -> Cronitor
